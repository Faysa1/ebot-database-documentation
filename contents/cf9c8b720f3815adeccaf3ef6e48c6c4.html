Ebot | Chatbot para ecommerce

Description

Uma grande necessidade dos consumidores hoje é tirar suas dúvidas antes de comprar um produto na internet. Da mesma forma o lojista deve preparar um profissional para atender esse tipo de necessidade.

Para manter o cliente sempre engajado surge o E-bot, que visa manter o usuário sempre engajado sem a necessidade de uma pessoa fazendo isso.

Mas não é só o usuário que ganha com isso, o lojista também terá grandes vantagens enquanto automatiza seu processo, veja a seguir:

Usuário | O usuário terá dúvidas sanadas rapidamente, pois o e-bot vai:

  • Garantir um atendimento personalizado e humanizado;
  • Diminuir o tempo de espera;
  • Apresentar detalhes do produto;
  • Indicar produtos mais comprados;
  • Cadastro;
  • Compras;

Lojista | O lojista também terá diversas vantagens, tais como:

  • Melhorar a experiência do seu usuário;
  • Aumento de vendas;
  • Dados de produtos mais vendidos;
  • Dados de produtos mais buscados;

Como a Ebot vai lucrar com esse serviço?

  • Pra cada venda realizada dentro do serviço será cobrado uma porcentagem sobre o valor;
  • Para cada clique que redirecione para a página do produto será cobrado um valor fixo;

Por isso é importante salvar todos os dados, para que eventualmente seja debitado o valor correto.

Exemplo de interação do usuário:

  • [Usuário]: Olá
  • [Bot]: Olá, como posso ajudá-lo?
  • [Usuário]: Quero comprar uma camiseta
  • [Bot]: Qual o seu tamanho?
  • [Usuário]: M ...

A cada interação com o chatbot ajudará o usuário a especificar ainda mais os resultados de busca, cada interação será uma query ao banco.
